# File-Upload-Data-Display

Upload a file and display data in that file

# Getting started

- Navigate to apps/frontend
- run npm install
- run npm run dev

- navigate to apps/backend
- run npm install
- run npm run dev

after runnifn both, navigate to `http://localhost:5173/`

Upload a file and display data in that file
